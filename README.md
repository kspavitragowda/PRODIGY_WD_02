# PRODIGY_WD_02

Stopwatch Web Application

The Stopwatch Web Application is a responsive and interactive timer built using HTML, CSS, and JavaScript. It allows users to accurately measure elapsed time, pause and resume the timer, reset the stopwatch, and record lap times with millisecond precision.

Features

- Start the stopwatch
- Pause and resume functionality
- Reset the stopwatch
- Millisecond precision timing
- Lap time recording
- Responsive design
- Modern and user-friendly interface

How It Works

1. Click Start to begin timing.
2. Click Pause to temporarily stop the timer.
3. Click Resume to continue from the paused time.
4. Click Lap to record the current elapsed time.
5. Click Reset to clear the timer and lap history.

The stopwatch displays time in hours, minutes, seconds, and milliseconds, ensuring accurate time measurement.
